Python Reel Theme
=================

1. Слоган. Міняється через Diazo theme панель (змінна slogan)
2. Тема має можливість зміни фону для топу. Для цього потрібно додати зображення з id topimage
3. Тема має альтернативний топ, який використовує білий колір тексту. Він передбачений для варіанту коли для топу буде використовуватись темний малюнок. Щоб активувати альтернативний топ, потрібно змінній top присвоїти значення white.
4. У топ можна вставити карусель. Для цього потрібно додати карусель і встановити для неї id top-carousel (у іншому випадку карусель буде в контенті). Також у цьому випадку можна змінити колір фону топу, призначивши змінній top_background інше значення, яке буде відповідати кольору фону.
5. У футері є 4 іконки для соц мереж. Лінки на них можна вставити через редагування темплейту футера через ZMI.
6. Можна змінювати ширину колонок і відстань між ними, для цього необхідно змінити значення columntwowidth, columnonewidth, columnsmargin.
7. Протестовано з такими продуктами: Products.Carousel, Products.ContentWellPortlets, Products.PloneFormGen, quintagroup.dropdownmenu, ducts.LinguaPlone
